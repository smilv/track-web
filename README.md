### react 练习项目

#### 目录结构

| api - 域名配置  
| build - 打包后目录  
| config - webpack 配置文件  
| src  
| —— assets - 静态资源  
| —— axios - 接口调用  
| —— components - 组件  
| —— mock - 假数据  
| —— views - 页面组件  
| —— route - 路由  
| —— utils - js 工具类  
| App.jsx - 主页组件  
| index.js - 主入口文件

#### 命令行

npm start 本地开发  
npm run build 打包编译

#### 开发建议

-   使用 jsx 作为 React 组件的扩展名
-   组件文件命名规则：KebabCase.jsx
-   js、scss 文件命名规则：kebab-case.js/scss
-   img 文件命名规则：kebab_case.png
